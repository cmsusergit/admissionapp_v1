# College Authority User Guide

This guide details the processes for college staff responsible for verifying student applications and managing admissions.

## 1. Dashboard Overview

After logging in as a **College Authority**, you will see:
*   **Total Applications:** Number of applications received for your college.
*   **Pending Verification:** Applications waiting for review.
*   **Approved:** Applications that have passed verification.
*   **Rejected:** Applications deemed ineligible.

---

## 2. Application Verification

### Reviewing Applications
1.  Go to the **Applications** tab in the main navigation.
2.  Use the **Filters** to find "Submitted" or "Pending" applications.
3.  Click on an application ID to view details.

### Split-Screen Verification
The verification interface shows:
*   **Left Panel:** Student data (Name, Marks, etc.) entered in the form.
*   **Right Panel:** Uploaded documents corresponding to that data.

1.  **Personal Details:** Check the student's name, DOB, and category against the uploaded ID proof.
2.  **Academic Details:** Compare the entered marks with the uploaded marksheets.
3.  **Document Actions:**
    *   **Approve:** If the document is clear and matches the data.
    *   **Reject:** If the document is illegible, incorrect, or missing. Select a reason (e.g., "Blurry", "Wrong Year", "Name Mismatch").

### Final Decision
1.  **Approve Application:** If all documents and data are correct.
    *   The application moves to the **Approved** list.
2.  **Request Correction:** If any document/field was rejected.
    *   The student will be notified to re-upload.
    *   The application status changes to **Needs Correction**.
3.  **Reject Application:** If the student is ineligible (e.g., marks below cutoff).
    *   Provide a detailed reason.
    *   The application status changes to **Rejected**.

---

## 3. Merit List Generation (College Level)

If your college manages its own merit lists:
1.  Go to the **Merit Lists** tab.
2.  Select the **Admission Cycle** and **Course**.
3.  Click **Generate Merit List**.
    *   The system will calculate scores based on defined formulas.
4.  **Review:** Check the generated list for accuracy.
5.  **Publish:** Click **Publish** to make the list visible to students.

---

## 4. Admission Confirmation

Once a student accepts an offer and pays fees:
1.  Go to the **Admissions** tab.
2.  You will see a list of students with **Provisional Admission**.
3.  Verify the payment status (usually automated).
4.  Click **Confirm Admission** to finalize their enrollment.
5.  **Print Admission Letter:** You can download/print the official letter for college records.

---

## 5. Reports

1.  Go to the **Reports** section.
2.  Select the report type (e.g., "Daily Applications", "Category-wise Admissions").
3.  Choose the date range.
4.  Click **Download CSV** or **Print**.
