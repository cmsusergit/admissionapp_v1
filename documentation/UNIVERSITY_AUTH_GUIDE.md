# University Authority User Guide

This guide is for University-level administrators responsible for the entire admission process across multiple colleges.

## 1. Dashboard Overview

After logging in as a **University Authority**, you will see:
*   **Total Admissions:** Number of successful enrollments across all affiliated colleges.
*   **Verified Applications:** Number of applications pending approval.
*   **Merit List Schedule:** Upcoming dates for merit list generation.
*   **Reports:** Quick links to admission statistics.

---

## 2. Managing Admissions

### Admission Cycles
1.  Go to the **Admission Cycles** tab.
2.  Click **Create New Cycle**.
3.  Enter the details:
    *   Academic Year (e.g., 2026-27).
    *   Start Date: When applications open.
    *   End Date: When applications close.
4.  Click **Save**.
5.  **Publish Cycle:** Makes the cycle active for students.

### Course Management
1.  Go to the **Courses** tab.
2.  Click **Add New Course**.
3.  Enter course details (Name, Code, Intake Capacity, etc.).
4.  Link the course to specific colleges.
5.  Click **Save**.

---

## 3. Merit Rules & Lists (Global)

For university-wide merit lists:
1.  Go to the **Merit Rules** tab.
2.  **Define Formula:** Create a new formula for scoring.
    *   Select the criteria (e.g., Entrance Marks, HSC Marks).
    *   Set the weightage for each (e.g., Entrance: 60%, HSC: 40%).
    *   Save the formula.
3.  **Map Formula:** Assign the formula to specific courses/colleges.

### Generating Merit Lists
1.  Go to the **Merit Lists** tab.
2.  Select the **Admission Cycle** and **Course**.
3.  Click **Run Merit Generation**.
    *   The system calculates scores for all verified applications.
    *   This may take a few minutes depending on the volume.
4.  **Review:** Check the top ranks for anomalies.
5.  **Publish:** Click **Publish** to release the list on the student portal.

---

## 4. Final Approvals

For applications requiring university approval:
1.  Go to the **Approvals** tab.
2.  Filter for "Pending University Approval".
3.  Review the application details and documents.
4.  **Approve:** Validate the application for final admission.
5.  **Reject/Return:** Send back to the college for correction.

---

## 5. Reports

1.  Go to the **Reports** section.
2.  Generate comprehensive reports:
    *   **Admission Summary:** Total applications, verified, admitted.
    *   **College-wise Report:** Admissions per college.
    *   **Category-wise Report:** Break down by reservation category.
    *   **Fee Collection Report:** Total fees collected.
3.  Export data to Excel/PDF.
