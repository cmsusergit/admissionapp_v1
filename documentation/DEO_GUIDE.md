# Data Entry Operator (DEO) User Guide

This guide details the process for Data Entry Operators (DEOs) applying on behalf of students.

## 1. Apply on Behalf

### Starting an Application
1.  Go to the **Applications** tab in the main navigation.
2.  Click **New Application**.
3.  Select the **Admission Cycle** and **Course**.
4.  Enter the Student's Full Name (as per documents).
5.  Click **Create Draft**.

### Filling the Application
1.  **Personal Details:** Enter the student's personal information (DOB, Gender, Category, Address, Mobile).
    *   *Note:* Ensure the mobile number is correct for notifications.
2.  **Academic History:** Enter details of previous qualifications.
    *   Verify marks against the original marksheets.
3.  **Document Upload:**
    *   Scan the physical documents provided by the student.
    *   Upload each file (Photo, Signature, Marksheets, etc.).
    *   Ensure clarity and correct file format.
4.  **Review:** Check all entered data with the student present (if possible).
5.  **Submit Application:** Click **Submit**.

---

## 2. Managing Drafts

If an application is incomplete:
1.  Go to the **My Drafts** tab.
2.  Find the application by student name or draft ID.
3.  Click **Resume Application**.
4.  Complete the missing sections.
5.  **Submit** when ready.

---

## 3. Search & View

To find a previously submitted application:
1.  Go to the **Search** tab.
2.  Enter the Student Name or Application ID.
3.  View the application status (Submitted, Verified, etc.).
4.  **Print Application Form:** Click to generate a PDF for the student's record.

---

## 4. Reports

1.  Go to the **Reports** section.
2.  View the number of applications submitted by you today/this week.
3.  Check for any errors or rejections flagged by the system.
